Submission Guidelines

Follow these guidelines to ensure a successful code submission:

  • Choose the correct programming language for your code.
  • Select the problem you are solving from the dropdown list.
  • If using the Code Editor, write your code in the provided space. If using File Upload, submit your code as a file.
  • Click the "Submit" button to submit your code for grading.
